## 简介

clone from `Saidmin Vue` 3.0
### 计划
把在花瓣和UI中国看到的一些好看的设计稿简单实现一下，做成合集。一方面为了提升能力，一方面给自己生成一份作品集。包括：
- 工作台样式、后台管理菜单样式（主题切换）、CSS特效、学习笔记等等。

## 安装使用

- 获取代码

```
git clone https://gitee.com/appsai/saiadmin-vue.git
```


- 安装依赖

```
cd saiadmin-vue && yarn install
```

- 运行

```
yarn dev
```

- 打包

```
yarn build
```
